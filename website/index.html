Project Description

For my final project I wanted to implement soft shadows, and to try out some noise generations on some simple scenes.

Noise Functions Used:

  • Perlin Noise
  • Simplex Rigid Noise
  • Warped Curl of Simplex Noise
  • 1-D Simplex Noise Fractal Brownian Motion Sum with Analytical Derivatives
  • 1-D Simplex Ridged Multi-Fractal Noise Sum
